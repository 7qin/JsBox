# JsBox
[点击预览](http://code.smallcfj.club)

采用Vue+nodejs实现的在线Js编辑器，前后台通过接口进行访问，可以在线编辑、保存、分享Js代码，实现了简单的响应式。
前台：vue
后台：nodejs、文件存储
